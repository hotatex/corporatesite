## 5. 技術仕様

### 5.1 開発環境
- **フレームワーク**: Next.js 14
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **アイコン**: React Icons

### 5.2 ビルド設定
- **出力形式**: 静的サイト（next export）
- **画像最適化**: 無効化（Cloudflare Pages対応）
- **トレーリングスラッシュ**: 有効
