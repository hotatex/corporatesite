# Tailwinds Corporate Site

株式会社Tailwindsのコーポレートサイトです。

## 技術スタック

- Next.js 14
- TypeScript
- Tailwind CSS
- React Icons

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド（静的サイト生成）
npm run build
```

## デプロイ

Cloudflare Pagesにデプロイします。

- ビルドコマンド: `npm run build`
- ビルド出力ディレクトリ: `out`
- 環境変数: `NODE_VERSION=18`
